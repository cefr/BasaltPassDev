User & Admin Guide

Manuals for End Users, Tenant Admins, and Platform Admins.

📄️ Tenant Administration

In BasaltPass, a Tenant is an isolated environment for your organization or project. As a Tenant Admin, you have full control over the users, roles, and configuration within your tenant.

📄️ Managing Users

As a Tenant Admin, you can invite, manage, and remove users from your tenant.

📄️ Managing Roles & Permissions

BasaltPass uses a flexible RBAC (Role-Based Access Control) system.

📄️ Password Policies

Admins can configure password complexity requirements to enhance security.

📄️ Two-Factor Authentication (2FA)

BasaltPass supports TOTP (Time-based One-Time Password) for 2FA.

📄️ Audit Logs

BasaltPass records significant actions for security and compliance.

📄️ Forgot Password

The self-service password reset flow allows users to regain access without admin intervention.

📄️ Profile Management

Users can manage their personal information via the User Console.

📄️ Coupon System

The Tenant Coupon System allows organizations to manage their own discounts and promotions independently.

📄️ Payments (Stripe)

This document explains the current BasaltPass payment flow using tenant-level Stripe configuration.

📄️ Platform Administration

Platform Admin APIs are used for managing the entire BasaltPass instance. These are distinct from Tenant Admin APIs.
